Git & GitHub CheatSheet

            
Commands

            git --version                            To check git version
            git config --global user.name “username” To Enter Username while setup
            git config --global user.email “useremail” To Enter User Email while setup
            git config --list                        To Check User details
            git clone <repo link>                  To clone a GitHub repository
            git status                               Check the status of code
            git add <filename>                    Add a specific file for staging
            git add .                                 Add all files for staging
            git commit -m “message”                  To commit a change
            git push origin main                     Push changes to main branch
            git push origin <branch>                 Push changes to a specific branch
            git log                                  Getting git history

            
Note: File markers meaning in VS Code

            M -> Modified (any change made in existing file)
            U -> Untracked (a file is not yet tracked by git)
            A -> Added File (file added at staging, but, not committed yet)

            
Create a new git repository in your local system

            git init                                 Set git in a local repository
            git remote add origin <link>             Transfer origin from local system to a GitHub repository
            git remote -v                            Verify the remote git
            git branch                               Check the branch working on
            git branch -M main                       Rename branch name from master to main
            git push origin main                     Push changes to github
            git push -u origin main (u -> upstream), using ‘git push’ works fine next time, if working on same branch

            
Branches

            git branch                               Check current working branch
            git branch -M <name>                     Rename branch
            git checkout <name>                      Navigate to another branch
            git checkout -b <name>                   Create a new branch
            git checkout -d <name>                   Delete a branch
            git diff <name>                         Compare current branch with another branch
            git merge <name>                        Merge two branches
            git pull origin <name>                   Getting changes from GitHub to Local System

            
Undo Changes

            git reset <filename>                    At staged (particular file)
            git reset                                 At staged (all files)
            git reset HEAD~1                         Undo single commit
            git reset <hash>                         Multiple commit back (inside git)
            git reset --hard <hash>                    Multiple commit back (inside vs code)
